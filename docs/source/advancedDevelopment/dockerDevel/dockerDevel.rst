Developing with docker and with the Frank!Flow
==============================================

In :ref:`gettingStarted`, you learned the basics of developing Frank configurations. You used the Frank!Runner (https://github.com/wearefrank/frank-runner) to run them. This section explains how you can develop without the Frank!Runner and using docker. It also explains how you can use the Frank!Flow, a tool to edit Frank configurations graphically.
